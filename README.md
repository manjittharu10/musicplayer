# musicplayer
A custom music player built using HTML, CSS, and JavaScript, fetching real-time music data from the Deezer API. This project demonstrates API integration, audio playback, and interactive UI features.
 Features:

. Fetches trending tracks dynamically from the Deezer API

. Play, pause, and switch tracks seamlessly

. Interactive and responsive user interface

. Built with pure JavaScript, HTML, and CSS (no frameworks)

 Tech Stack:

. HTML

. CSS

. JavaScript (Vanilla JS)

. Deezer API

 What I Learned:

. Consuming and handling data from a third-party API

. Working with the HTML5 Audio API

. Building dynamic and interactive frontend features

. DOM manipulation, event handling, and asynchronous JavaScript (fetch, async/await)

Future Improvements:

. Add track search functionality

. Playlist management

. Improved UI/UX with animations

. Mobile-first responsive design
